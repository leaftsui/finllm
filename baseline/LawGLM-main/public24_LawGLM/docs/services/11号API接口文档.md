## 11 号API接口函数 根据法院名称或者法院代字查询法院等相关数据。
``` 
def get_court_code_service(court_name: Annotated[str, "法院名称\法院代字", True],need_fields: Annotated[str, "需要返回的字段列表 例如[法院名称,法院代字,行政级别]", True]) -> str:
    """
    根据法院名称或者法院代字查询法院级别和代号等相关数据。

    参数:
    court_name -- 法院名称或者法院代字，例如"上海市浦东新区人民法院"或者"沪0115"
    need_fields -- 需要返回的字段列表，例如[法院名称,行政级别,法院级别,法院代字,区划代码,级别],need_fields传入空，则表示返回所有字段，否则返回填入的字段

    例如：
        输入：
        {"法院名称": "上海市浦东新区人民法院"}
        输出：
        {'法院名称': '上海市浦东新区人民法院', 
        '行政级别': '市级', 
        '法院级别': '基层法院', 
        '法院代字': '沪0115', 
        '区划代码': '310115', 
        '级别': '1'}
    """
``` 
### 例子： 
### 法院名称
##### 输入
``` 
{"法院名称": "上海市浦东新区人民法院", "need_fields":["法院名称","行政级别","法院级别","法院代字","区划代码","级别"]}
``` 
#### 输出 
``` 
{'法院名称': '上海市浦东新区人民法院',
 '行政级别': '市级',
 '法院级别': '基层法院',
 '法院代字': '沪0115',
 '区划代码': '310115',
 '级别': '1'}

```
### 法院代字
#### 输入
``` 
{"法院代字": "吉24","need_fields":[] }
``` 
#### 输出
``` 
{'法院名称': '延边朝鲜族自治州中级人民法院',
 '行政级别': '市级',
 '法院级别': '中级法院',
 '法院代字': '吉24',
 '区划代码': '222400',
 '级别': '1'}
```
