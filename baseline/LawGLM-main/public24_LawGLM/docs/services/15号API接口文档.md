## 15号API接口函数 	根据省，市，区一同查询区划代码
``` 
def get_address_code_service(province: Annotated[str, "省份", True],city: Annotated[str, "城市", True],county: Annotated[str, "区县", True],need_fields: Annotated[str, "需要返回的字段列表", True] ) -> str:
    """
    根据省，市，区一同查询区划代码。

    参数:
    province -- 省份，例如"西藏自治区"
    city -- 城市，例如"拉萨市"
    county -- 区县，例如"城关区"
    need_fields -- 需要返回的字段列表，例如[省份,城市,城市区划代码,区县,区县区划代码],need_fields传入空，则表示返回所有字段，否则返回填入的字段

    例如：
        输入：
        province = "西藏自治区",city = "拉萨市",county = "城关区"
        输出：
        {'省份': '西藏自治区', 
        '城市': '拉萨市', 
        '城市区划代码': '540100000000', 
        '区县': '城关区', 
        '区县区划代码': '540102000000'}
    """

```
