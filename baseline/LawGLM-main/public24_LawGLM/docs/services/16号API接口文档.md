## 16号API接口函数 	根据日期及省份城市查询天气相关信息。
``` 
def get_temperature_info_service(province: Annotated[str, "省份", True],city: Annotated[str, "城市", True],date: Annotated[str, "日期", True],need_fields: Annotated[str, "需要返回的字段列表", True] ) -> str:
    """
   根据日期及省份城市查询天气相关信息。

    参数:
    province -- 省份，例如"北京市"
    city -- 城市，例如"北京市"
    date -- 日期，例如"2020年1月1日"
    need_fields -- 需要返回的字段列表，例如[日期,省份,城市,天气,最高温度,最低温度,湿度],need_fields传入空，则表示返回所有字段，否则返回填入的字段

    例如：
        输入：
        province = "北京市",city = "北京市",date = "2020年1月1日"
        输出：
        {'日期': '2020年1月1日', 
        '省份': '北京市', 
        '城市': '北京市', 
        '天气': '晴', 
        '最高温度': '11', 
        '最低温度': '1', 
        '湿度': '55'}
    """
``` 

### 例子： 
### 1
#### 输入
``` 
'省份': "四川省", 
'城市': "成都市", '日期': "2019年12月11日"
``` 
#### 输出 
``` 
'{"日期": "2019年12月11日", "省份": "四川省", "城市": "成都市", "天气": "晴", "最高温度": "16", "最低温度": "1", "湿度": "55"}'
```
